## Deskripsi Praktikum

Praktikum Basis Data P7 ini membahas tentang penggunaan **View** dalam sistem basis data. Pada praktikum ini mahasiswa mempelajari cara membuat view menggunakan perintah `CREATE VIEW AS SELECT`, menampilkan struktur view dengan `DESCRIBE VIEW`, menghapus view menggunakan `DROP VIEW`, serta melakukan pembaruan data pada view melalui `UPDATE VIEW`.

Melalui praktikum ini, mahasiswa diharapkan mampu memahami konsep view sebagai tabel virtual yang digunakan untuk mempermudah pengelolaan dan penyajian data tanpa mengubah tabel asli pada database. Selain itu, praktikum ini juga bertujuan meningkatkan kemampuan mahasiswa dalam mengelola database menggunakan perintah SQL secara efektif dan efisien.

Materi yang dipraktikkan dalam laporan ini meliputi:

* Create View As Select Table
* Describe View
* Drop View
* Update View

Praktikum ini disusun sebagai bagian dari pembelajaran mata kuliah Basis Data di Politeknik Purbaya Tegal tahun ajaran 2025/2026.
